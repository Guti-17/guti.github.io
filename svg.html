SVG Graphics

<circle> Element

The <circle> element draws a circle in SVG:

<svg width="100" height="100">
  <circle cx="50" cy="50" r="40" fill="blue" />
</svg>

<rect> Element

The <rect> element draws a rectangle in SVG:

<svg width="200" height="100">
  <rect x="10" y="10" width="150" height="80" fill="green" />
</svg>

<line> Element

The <line> element draws a straight line in SVG:

<svg width="200" height="100">
  <line x1="0" y1="0" x2="200" y2="100" stroke="red" stroke-width="2" />
</svg>

<polygon> Element

The <polygon> element draws a shape with multiple sides:

<svg width="200" height="200">
  <polygon points="50,150 150,150 100,50" fill="purple" />
</svg>

<path> Element

The <path> element allows creating complex shapes in SVG:

<svg width="200" height="200">
  <path d="M10 80 C 40 10, 65 10, 95 80 S 150 150, 180 80" fill="transparent" stroke="orange" stroke-width="4" />
</svg>
