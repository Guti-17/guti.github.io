CSS Properties

`color` Property

Sets the text color:

p { color: red; }

`background-color` Property

Sets the background color:

body { background-color: #f0f0f0; }

`font-size` Property

Sets the font size of the text:

h1 { font-size: 32px; }

`margin` Property

Sets the margin of an element:

div { margin: 20px; }

`padding` Property

Sets the padding inside an element:

p { padding: 10px; }
